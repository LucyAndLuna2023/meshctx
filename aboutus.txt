[ENGLISH]

MeshCtx is an open-source multi-agent framework built on the principles of 
"Division of Labor + Full Transparency." It's not a black-box single agent — 
it's a mesh network of an Orchestrator, a Guardian, and specialized agents 
working together under full human supervision.

WHY MeshCtx?

Current agents (OpenClaw, Hermes, etc.) proved "AI can work," but exposed 
three fatal flaws:

  1. UNCONTROLLED — Deleting files, crashing systems, flagged by regulators
  2. BLACK BOX — You never know why it did what it did
  3. UNTRACEABLE GROWTH — Self-evolution without audit trails means 
     enterprises can't trust it

MeshCtx redesigns from the ground up:

  ✅ AUDITABLE — Every call, every decision has an execution report; 
     traceable like Git history
  ✅ INTERCEPTABLE — Critical operations (delete, external data sending) 
     require user or Guardian confirmation
  ✅ CONTROLLABLE EVOLUTION — All skill and strategy changes are versioned; 
     roll back anytime
  ✅ COST-EFFICIENT — Multi-tier caching reduces API calls by 60%+
