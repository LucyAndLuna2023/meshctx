Why meshctx?

🧠

Hierarchical Memory

4-tier memory system (L0-L4) modeled after human cognition. With the Ebbinghaus Forgetting Curve — important info persists, trivia naturally decays.

🔄

Self-Evolving

Meta-cognition loop runs after every task: evaluate → extract patterns → update knowledge graph → adjust behavior. Gets smarter every time.

🎭

Multi-Agent Orchestra

One intent. Automatically decomposed into a task DAG, executed in parallel by specialized agents (Coder + Researcher + DevOps + Reviewer).

🔮

Predictive Context

Learns your patterns. Pre-loads relevant memories before you ask. Context assembly in under 50ms.

🔌

Plugin Marketplace

MCP protocol native. pip install meshctx-plugin-xxx. Community-driven ecosystem of tools and integrations.

🔓

100% Open Source

MIT license. Free forever. Self-host on your own infrastructure with Docker or Kubernetes. No vendor lock-in.

About meshctx

🇨🇳 Built in China, Open to the World

meshctx is an open-source multi-agent framework built on "Division of Labor + Full Transparency." It's not a black-box single agent — it's a mesh network of an Orchestrator, a Guardian, and specialized agents working under full human supervision.

  • Auditable — Every call, every decision has an execution report
  • Interceptable — Critical operations require user or Guardian confirmation
  • Controllable Evolution — All skill changes are versioned; roll back anytime
  • Cost-Efficient — Multi-tier caching reduces API calls by 60%+

🇨🇳 中国开源，服务全球

meshctx 是"分工协作 + 完全透明"的中国开源多智能体框架。非单一黑盒 Agent，而是由编排器、监管员与专业智能体组成的网状协作系统。

  • 可审计 — 每次调用、每次决策都有执行报告，像 Git 一样可回溯
  • 可拦截 — 删除、外发等关键操作需用户或监管员确认
  • 可控进化 — 技能和策略变更全版本化，随时回滚
  • 成本友好 — 四层缓存，API 调用降低 60% 以上
