About MeshCtx

🌍 Open Source, Global Community

MeshCtx is an open-source multi-agent framework built on "Division of Labor + Full Transparency." — a mesh network of an Orchestrator, Guardian, and specialized agents under full human supervision. It's not a black-box single agent — it's a mesh network of an Orchestrator, a Guardian, and specialized agents working under full human supervision.

  • Auditable — Every call, every decision has an execution report
  • Interceptable — Critical operations require user or Guardian confirmation
  • Controllable Evolution — All skill changes are versioned; roll back anytime
  • Cost-Efficient — Multi-tier caching reduces API calls by 60%+

🌍 全球开源社区

MeshCtx 是基于"分工协作 + 完全透明"的开源多智能体框架 — 由编排器、监管员与专业智能体组成的网状协作系统。

  • 可审计 — 每次调用、每次决策都有执行报告，像 Git 一样可回溯
  • 可拦截 — 删除、外发等关键操作需用户或监管员确认
  • 可控进化 — 技能和策略变更全版本化，随时回滚
  • 成本友好 — 四层缓存，API 调用降低 60% 以上

🌍 グローバル・オープンソース

MeshCtx は 「分業＋完全透明性」を基盤とするオープンソースのマルチエージェントフレームワーク — オーケストレーター、ガーディアン、専門エージェントのメッシュネットワーク。

  • コスト効率 — マルチレベルキャッシュでAPI呼出60%削減
