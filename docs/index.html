30秒快速开始

268|
269| # 安装
270| git clone https://github.com/LucyAndLuna2023/meshctx.git
271| cd meshctx && pip install -e . && pip install pycryptodome

272| # 配置 DeepSeek（免费）
273| export DEEPSEEK_API_KEY=sk-你的key

274| # 扫描模型 & 开始聊天
275| meshctx model scan
276| meshctx chat

277| # 或打开浏览器 Web Chat
278| meshctx start && open http://localhost:3000/ui/chat 279|

🪟 Windows 用户 (WSL)

287|
288| # 方法1: 一键安装 (推荐)
289| 下载 meshctx.exe → 双击运行，输入API Key

291| wsl --install # 先装WSL(如未安装)
292| wsl bash -c "curl -fsSL https://raw.githubusercontent.com/LucyAndLuna2023/meshctx/main/install.sh | bash"

293| # 启动
294| wsl meshctx start
295| 浏览器打开 http://localhost:3000/ui/chat 296|
