About meshctx

🇨🇳 Built in China, Open to the World

meshctx is an open-source multi-agent framework built on "Division of Labor + Full Transparency." It's not a black-box single agent — it's a mesh network of an Orchestrator, a Guardian, and specialized agents working under full human supervision.

  • Auditable — Every call, every decision has an execution report
  • Interceptable — Critical operations require user or Guardian confirmation
  • Controllable Evolution — All skill changes are versioned; roll back anytime
  • Cost-Efficient — Multi-tier caching reduces API calls by 60%+

🇨🇳 中国开源，服务全球

meshctx 是"分工协作 + 完全透明"的中国开源多智能体框架。非单一黑盒 Agent，而是由编排器、监管员与专业智能体组成的网状协作系统。

  • 可审计 — 每次调用、每次决策都有执行报告，像 Git 一样可回溯

  • 成本友好 — 四层缓存，API 调用降低 60% 以上

💻 GitHub

30秒快速开始

# 安装
git clone https://github.com/LucyAndLuna2023/meshctx.git
cd meshctx && pip install -e . && pip install pycryptodome

# 配置 DeepSeek（免费）
export DEEPSEEK_API_KEY=sk-你的key

# 扫描模型 & 开始聊天
meshctx model scan
meshctx chat

# 或打开浏览器 Web Chat
meshctx start && open http://localhost:3000/ui/chat

🪟 Windows 用户 (WSL)

# 方法1: 一键安装 (推荐)
下载 meshctx.exe → 双击运行，输入API Key

# 方法2: 手动安装
wsl --install # 先装WSL(如未安装)
wsl bash -c "curl -fsSL https://raw.githubusercontent.com/LucyAndLuna2023/meshctx/main/install.sh | bash"

# 启动
wsl meshctx start
浏览器打开 http://localhost:3000/ui/chat

💡 meshctx 通过 WSL 在 Windows 上运行，可以访问全部 Windows 文件 (C:\ → /mnt/c/ E:\ → /mnt/e/)
