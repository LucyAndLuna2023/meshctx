About MeshCtx

🇨🇳 Built in China, Open to the World

MeshCtx is an open-core multi-agent framework built on "Division of Labor + Full Transparency." It's not a black-box single agent — it's a mesh network of an Orchestrator, a Guardian, and specialized agents working under full human supervision.

  • Auditable — Every call, every decision has an execution report
  • Interceptable — Critical operations require user or Guardian confirmation
  • Controllable Evolution — All skill changes are versioned; roll back anytime
  • Cost-Efficient — Multi-tier caching reduces API calls by 60%+

🇨🇳 中国研发，服务全球
